# 📊 IPL Analysis Dashboard (2008–2024)


---

## 🎯 Purpose of the Dashboard

This dashboard is created to analyze and visualize IPL (Indian Premier League) data from 2008 to 2024.

It helps in:
- Understanding match performance and outcomes  
- Analyzing team and player statistics  
- Identifying scoring patterns and trends  
- Supporting data-driven insights in sports analytics  

📄 Reference Report: IPL Analysis Dashboard  

---

## 🧱 Dashboard Structure

### 📌 PART 1: Primary KPI Section

**What we are making:**  
Key performance indicators (KPIs) to give a quick summary.

**KPIs:**
- Total Matches  
- Total Runs  
- Total Wins  
- Win Percentage  
- Total Wickets  
- Highest Team Score  

**Visuals Used:**
- Card visuals  
- KPI tiles  
- Icons  

---

### 🏏 PART 2: Match Analysis

**Includes:**
- Teams and scores  
- Toss winner & decision  
- Venue  
- Match type  
- Winner  

**Example:**
- MI vs KXIP (2020) → Tie + Double Super Over  

**Visuals:**
- Cards  
- Text visuals  
- Filters  

---

### 📈 PART 3: Running Total Analysis

**What it shows:**
- Cumulative runs over each over  

**Visuals:**
- Line chart  
- X-axis: Overs  
- Y-axis: Runs  

---

### ⚡ PART 4: Runs Per Over Analysis

**What it shows:**
- Runs scored per over  

**Visuals:**
- Clustered bar chart  

**Insight:**
- High overs → Power hitting  
- Low overs → Strong bowling  

---

### 🏆 PART 5: IPL Winners Analysis

**What it shows:**
- Titles won by teams  

**Top Teams:**
- Mumbai Indians → 5 Titles  
- Chennai Super Kings → 5 Titles  
- Kolkata Knight Riders → 3 Titles  

---

### 📊 PART 6: Team Performance

**Includes:**
- Matches  
- Wins  
- Batting average  
- Bowling average  
- Boundaries  

**Example:**
- CSK (2008):  
  - Wins: 9  
  - Avg: 25.89  

---

### 📋 PART 7: Points Table

**Includes:**
- Matches  
- Wins/Losses  
- Points  
- Net Run Rate  

**Example:**
- 2024 Winner: Kolkata Knight Riders  

---

### 👤 PART 8: Player Analysis

**Includes:**
- Runs  
- Strike Rate  
- Average  
- Matches  
- 4s & 6s  

**Example:**
- Virat Kohli → 165 runs  

---

### 🎛️ PART 9: Filter Panel

**Filters:**
- Year  
- Date  
- Match Number  
- Team  
- Player  

**Visuals:**
- Slicers  
- Dropdown filters
